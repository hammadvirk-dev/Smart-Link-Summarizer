# 🚀 Smart-Link-Summarizer
**Stop wasting time on long-form articles. Get the juice in seconds.**
Smart-Link-Summarizer is a lightweight Python tool that transforms any URL into a high-impact social media title, a concise executive summary, and actionable takeaways using **Google Gemini AI**.
## ✨ Why use this?
In an age of information overload, speed is your greatest competitive advantage.
 * **For Researchers:** Quickly vet sources without reading 2,000+ words.
 * **For Marketers:** Generate instant LinkedIn/Twitter hooks from your blog posts.
 * **For Content Creators:** Turn deep-dives into snackable newsletters in one click.
 * **For Everyone:** Save hours of reading every week.
## 🛠️ Features
 * 🕵️ **Smart Scraping:** Intelligent extraction of main article text, ignoring ads and navbars.
 * 🧠 **AI-Powered:** Uses gemini-2.5-flash for high-speed, high-accuracy analysis.
 * 🔗 **Viral Hooks:** Automatically generates click-bait style titles for maximum engagement.
 * 📋 **Structured Output:** Get clean Markdown-ready bullet points.
## 🚀 Quick Start
### 1. Clone the Repo
```bash
git clone [https://github.com/yourusername/Smart-Link-Summarizer.git](https://github.com/yourusername/Smart-Link-Summarizer.git)
cd Smart-Link-Summarizer

```
### 2. Install Dependencies
```bash
pip install -r requirements.txt

```
### 3. Set Your API Key
Export your Google Gemini API Key:
```bash
export GOOGLE_API_KEY='your_api_key_here'

```
### 4. Run the Script
```bash
python main.py

```
